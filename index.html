Sesión 3: Primer scraping con BeautifulSoup

Instala las bibliotecas necesarias:

pip install requests beautifulsoup4
import requests
from bs4 import BeautifulSoup

url = "https://quotes.toscrape.com/"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

quotes = soup.find_all('span', class_='text')
for quote in quotes:
    print(quote.text)

Sesión 4: Guardar datos en CSV

import csv

quotes = soup.find_all('span', class_='text')
authors = soup.find_all('small', class_='author')

with open('quotes.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.writer(f)
    writer.writerow(['Cita', 'Autor'])
    for quote, author in zip(quotes, authors):
        writer.writerow([quote.text.strip(), author.text.strip()])

Sesión 5: Paginación automática

page = 1
while True:
    url = f"https://quotes.toscrape.com/page/{page}/"
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')

    quotes = soup.find_all('span', class_='text')
    if not quotes:
        break

    for quote in quotes:
        print(quote.text)

    page += 1

Nota: Siempre revisa los términos de uso del sitio antes de hacer scraping.

Curso de Web Scraping con Octoparse

Sesión 1: ¿Qué es Octoparse?

Octoparse es una herramienta visual para hacer scraping sin programar. Permite seleccionar elementos, extraer datos y automatizar tareas fácilmente.

Descárgalo desde: octoparse.com

Sesión 2: Crear tu primera tarea

  1. Abre Octoparse y selecciona "Modo rápido".
  2. Ingresa una URL, por ejemplo: https://books.toscrape.com
  3. Octoparse detectará automáticamente los datos en la página.
  4. Elige los elementos que deseas extraer (títulos, precios, etc.).
  5. Haz clic en "Extraer datos".

Sesión 3: Exportar los datos

  • Haz clic en "Exportar"
  • Elige el formato (CSV, Excel, JSON, etc.)
  • Guarda los datos localmente

Sesión 4: Uso de plantillas

Octoparse ofrece plantillas prediseñadas para sitios populares como Amazon, Twitter, LinkedIn:

  • Haz clic en "Plantillas"
  • Selecciona la plataforma
  • Ingresa tus parámetros (palabras clave, categoría, ciudad, etc.)
  • Ejecuta la tarea y obtén los resultados

Sesión 5: Consideraciones éticas y legales

Antes de hacer scraping, revisa los términos de uso del sitio web. No extraigas datos personales sin permiso y evita sobrecargar servidores con muchas solicitudes.
