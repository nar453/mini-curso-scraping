Sesión 3: Primer scraping con BeautifulSoup

Instala las bibliotecas necesarias:

pip install requests beautifulsoup4

Ejemplo básico:

import requests
from bs4 import BeautifulSoup

url = "https://quotes.toscrape.com/"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

quotes = soup.find_all('span', class_='text')
for quote in quotes:
    print(quote.text)

Sesión 4: Guardar datos en CSV

Extrae autores y guarda los datos:

import csv

quotes = soup.find_all('span', class_='text')
authors = soup.find_all('small', class_='author')

with open('quotes.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.writer(f)
    writer.writerow(['Cita', 'Autor'])
    for quote, author in zip(quotes, authors):
        writer.writerow([quote.text.strip(), author.text.strip()])

Sesión 5: Paginación automática

Recorre todas las páginas disponibles:

page = 1
while True:
    url = f"https://quotes.toscrape.com/page/{page}/"
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')

    quotes = soup.find_all('span', class_='text')
    if not quotes:
        break

    for quote in quotes:
        print(quote.text)

    page += 1
